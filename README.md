# Learn Evening Program

A premium evening learning center landing page for Bugesera, Rwanda. Focusing on discipline, quality teaching, and real results. Clean, luxurious, and organized design for maximum focus.

## Features Included

1. **Hero Section**: Clear messaging positioning the program as premium.
2. **About Section**: Simple breakdown of the program and its target audience.
3. **Courses Offered**: Visual presentation of Mathematics, Biology, Chemistry, English, and ICT Basics.
4. **Smart Feature**: Highlighted "Performance Tracking" to enhance credibility.
5. **Schedule**: Clear Monday - Friday, 6:00 PM – 9:00 PM timing block.
6. **Why Choose Us**: Value proposition on small classes, qualified teachers, exam focus.
7. **Testimonials**: Social proof to build trust.
8. **Contact & Registration**: WhatsApp direct link, phone numbers, and an embedded lead capture form (no payment required upfront).
9. **Luxury Design System**: White negative space, `brandblue` for education trust, and `brandblack` for strong headings. Typography powered by Poppins Google Font.

## How to View

Simply open the `index.html` file in any modern web browser. No compilation or build tools required because it uses raw HTML styled with the Tailwind CSS unpkg CDN for absolute simplicity and portability.

To modify the styles, edit the classes directly within the HTML elements according to the TailwindCSS utility class system.
